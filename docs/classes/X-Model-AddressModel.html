AddressModel.php : 15

Japanese address lookup model.

Provides address lookup by postal code using bundled address data.

Usage:

$address = $this->AddressModel->getAddressByPostCode('100-0001');
// Returns: ['prefectureCode' => '13', 'address' => '東京都 千代田区 千代田']
